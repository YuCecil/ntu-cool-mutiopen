# NTU COOL 批量開啟工具

批量開啟多個 NTU COOL 課程頁面的輔助工具，適合課程管理人員使用。

## 功能

- 貼入多個 NTU COOL 課程 URL（每行一個），一鍵依序開啟所有課程
- 下拉選單選擇目標頁面（Modules、Grades、People、Assignments、Discussions 等）或輸入自訂後綴
- **交錯排序**：將清單前後兩半交錯排列，分散對伺服器的請求壓力
- 學號 / Email 搜尋模式：批量開啟各使用者的搜尋頁面
- 快速模式（可調整開啟間隔）

## 技術

- 純 HTML + Vanilla JavaScript + CSS Variables，單一 HTML 檔案
- 無後端，無需安裝套件

## 使用方法

直接開啟 `多開網頁.html`，無需任何設定。
